# Upcoming Calendar

The **Upcoming** tab is your personal release schedule, giving you a sneak peek into when your favorite manga will drop their next chapters.

## The Prediction System

This feature is directly powered by Yomic's [Smart Updates](./smart-updates.md) algorithm. 
Instead of checking random dates, the Upcoming calendar visualizes the predictions made by the engine.

When you open the Upcoming tab, you'll see a timeline categorized by dates (e.g., *Today*, *Tomorrow*, *This Week*, *Later*). 

Manga are placed into these categories based on their historical release patterns. For example:
- If a manga consistently releases a chapter every Friday, and today is Thursday, you'll see it listed under **Tomorrow**.
- If a manga is highly irregular, it might be pushed to the bottom under **Unscheduled** or **Later**.

## Benefits

1. **Anticipation**: You no longer need to constantly check your library to see if a chapter is out. The calendar tells you exactly when to expect it.
2. **Transparency**: It provides a transparent view into *why* Yomic is (or isn't) checking a specific manga for updates today. 

*Note: Because this system relies on historical data, newly added manga might take a few chapter releases before they accurately appear on the Upcoming calendar.*
