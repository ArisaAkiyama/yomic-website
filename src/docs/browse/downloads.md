# Downloads

Yomic allows you to download manga chapters directly to your local device for offline reading. 

## Downloading Chapters

You can initiate a download in several ways:
- **Single Chapter**: Click the download icon next to any chapter in the Manga Detail page.
- **Bulk Download**: Use the "Download All" option to queue multiple chapters at once.
- **Context Menu**: Right-click on a chapter to find additional download options.

When a chapter is successfully downloaded, a visual checkmark badge will appear next to it. 

## The Download Manager

You can monitor the status of your active downloads by navigating to the **Downloads** tab in the sidebar. 
The Download Manager features a robust engine that handles tasks efficiently:
- **Parallel Downloading**: Yomic downloads multiple images simultaneously to maximize your network speed.
- **Temporary Files**: Partial downloads are saved as `.part` files to prevent corrupted images. Once an image is fully downloaded, the `.part` extension is removed.
- **Auto-Retry**: If a network error occurs, the manager will attempt to recover the download automatically.

## Storage Location
By default, all chapters are saved in the `Yomic/Downloads` folder inside your system's AppData directory. You can manage or clear these files from the Storage Settings menu.
