# General FAQ

Here are answers to some of the most commonly asked questions about Yomic.

### What is Yomic?
Yomic is a free, open-source manga, webtoon, and comic reader designed for desktop platforms (Windows, macOS, and Linux). It provides a sleek and modern native UI for discovering, managing, and reading your favorite titles seamlessly.

### Where do the manga come from?
Yomic does not host any manga on its own servers. Instead, it acts as a customized browser that extracts data directly from third-party catalog websites using the **Extensions** system.

### How do I install new sources?
By default, Yomic does not come with any manga sources. You must install **Extensions** (plugins) that connect Yomic to specific websites.
1. Navigate to the **Extensions** tab in the sidebar.
2. Browse the list of available extensions.
3. Click **Install** on the source you want.
Once installed, the source will appear in your **Browse Source** tab!

### Can Yomic bypass regional blocks (Cloudflare / VPN issues)?
Yes! Yomic features native client-side TLS packet fragmentation (DPI bypass) and automatic User-Agent configuration to access sources seamlessly without requiring a third-party VPN or external proxy. We also support **DNS over HTTPS** (configurable in Settings) to bypass standard ISP domain blocks.
