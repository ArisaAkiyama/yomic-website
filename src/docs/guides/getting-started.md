# Getting Started

Welcome to Yomic! This guide will help you install and set up the application for the first time.

## Installation

1. Go to the [Download](/download/) page or our [GitHub Releases](https://github.com/ArisaAkiyama/yomic/releases).
2. Download the appropriate installer or archive for your operating system (Windows, macOS, or Linux).
3. Extract or install the application and launch it.

## Adding Content

Yomic does not come with any pre-installed manga sources. To start reading, you'll need to install an extension:

1. Open Yomic and click on the **Extensions** icon in the sidebar.
2. Browse the available extensions and click the **Download** button next to the one you want.
3. Once installed, go to **Browse Source** to search for manga and add them to your library.

## Navigation

- **Library:** Your collection of saved manga.
- **Updates:** Recent chapters for manga in your library.
- **History:** Your recently read manga and chapters.
- **Browse:** Search for new manga from your installed sources.
- **Extensions:** Manage your manga sources.
- **Settings:** Configure reader preferences, appearance, and more.
