# Extensions

Extensions allow you to read manga from various websites directly within Yomic.

## Installing Extensions

1. Navigate to the **Extensions** tab in the sidebar.
2. You will see a list of available extensions.
3. Click the **Download** button next to the extension you want to install.
4. Once installed, the extension will appear under the "Installed" section.

## Using Extensions

After installing an extension, you can:
- Go to the **Browse** tab and select the extension to view its popular or latest manga.
- Use the search bar in the **Browse** tab to find specific titles within that extension.
- Add manga from the extension to your **Library**.

## Managing Extensions

You can manage your installed extensions from the **Extensions** tab:
- **Uninstall:** Remove an extension you no longer need.
- **Verify Cloudflare:** Some sources require you to solve a Cloudflare challenge periodically. Click the verify icon if you are having trouble loading chapters from a specific source.
