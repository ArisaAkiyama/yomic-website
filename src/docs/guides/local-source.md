# Local Source

**Status:** 🚧 Currently in Development

Currently, Yomic acts as a centralized browser for online catalogs and allows you to download those chapters for offline reading. However, native support for importing your own `.cbz`, `.cbr`, `.zip`, or raw image folders as a dedicated "Local Source" extension is planned for a future update.

### How to read offline files currently?
If you have downloaded chapters using Yomic's [Download Manager](../browse/downloads.md), you can read them entirely offline! Yomic automatically detects if a chapter exists in your local `Yomic/Downloads` folder and will load the images directly from your hard drive without making any network requests.
