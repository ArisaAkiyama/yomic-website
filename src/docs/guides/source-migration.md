# Source Migration

Sometimes a manga extension (source) might shut down, change its domain, or simply stop working. When this happens, you don't want to lose your reading progress or have to manually search for your manga in a new source.

Yomic provides a built-in **Source Migration Service** to handle this seamlessly.

## How to Migrate a Manga

1. Navigate to your **Library**.
2. Locate the manga you want to move to a new source.
3. Open the Manga Detail page and select the **Migrate** option.
4. Yomic will ask you to select a target Extension (e.g., migrating from *Source A* to *Source B*).
5. Yomic will automatically search the target extension for a manga with a matching title and author.
6. Once a match is found and confirmed, Yomic will update the internal database.

### What gets migrated?
- The manga is permanently relinked to the new source URL.
- Your **Read History** (which chapters you've marked as read) is preserved, as long as the new source uses a similar chapter numbering scheme.
- Categories and library status remain intact.

### What doesn't get migrated?
- Downloaded chapters. Because images are sourced differently across websites, old downloads tied to the dead source will not work for the new source. You will need to re-download them.
