# Kebijakan Privasi

**Tanggal Berlaku:** 24 Juni 2026

## Gambaran Umum

Yomic ("kami") adalah aplikasi desktop sumber terbuka (open-source) untuk membaca manga, webtoon, dan komik. Kami berkomitmen untuk melindungi privasi Anda.

Kebijakan Privasi ini menjelaskan bagaimana kami mengelola data Anda ketika Anda menggunakan aplikasi Yomic.

## Pengumpulan Data

**Yomic tidak mengumpulkan, menyimpan, atau mengirimkan data pribadi apa pun.**

Aplikasi ini beroperasi sepenuhnya secara lokal di perangkat Anda. Semua data terkait pustaka, riwayat membaca, dan pengaturan disimpan secara lokal di media penyimpanan (hard drive) Anda.

## Aktivitas Jaringan

Aplikasi ini membuat permintaan jaringan hanya untuk tujuan berikut:
1. **Mengambil Konten:** Untuk mengunduh informasi manga, bab, dan gambar dari ekstensi/sumber yang telah Anda pasang dan pilih untuk digunakan secara mandiri.
2. **Memeriksa Pembaruan:** Untuk memeriksa repositori resmi GitHub untuk mendeteksi versi rilis aplikasi atau ekstensi baru.

Permintaan ini dilakukan secara langsung dari perangkat Anda ke server masing-masing. Yomic tidak memproksi lalu lintas data ini atau melacak aktivitas membaca Anda.
