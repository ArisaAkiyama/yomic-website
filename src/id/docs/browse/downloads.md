# Unduhan

Yomic memungkinkan Anda mengunduh chapter manga langsung ke perangkat lokal Anda untuk dibaca secara luring (*offline*). 

## Mengunduh Chapter

Anda dapat memulai pengunduhan dengan beberapa cara:
- **Chapter Tunggal**: Klik ikon unduh di sebelah chapter mana pun di halaman Detail Manga.
- **Unduhan Massal**: Gunakan opsi "Unduh Semua" untuk mengantrekan banyak chapter sekaligus.
- **Menu Konteks**: Klik kanan pada sebuah chapter untuk menemukan opsi pengunduhan tambahan.

Saat sebuah chapter berhasil diunduh, sebuah lencana centang visual akan muncul di sebelahnya. 

## Manajer Unduhan

Anda dapat memantau status unduhan aktif Anda dengan menavigasi ke tab **Unduhan** di *sidebar*. 
Manajer Unduhan dilengkapi dengan mesin tangguh yang menangani tugas secara efisien:
- **Pengunduhan Paralel**: Yomic mengunduh beberapa gambar secara bersamaan untuk memaksimalkan kecepatan internet Anda.
- **File Sementara**: Unduhan yang belum selesai (parsial) disimpan sebagai file `.part` untuk mencegah kerusakan gambar. Setelah gambar selesai diunduh sepenuhnya, ekstensi `.part` akan dihapus.
- **Coba Lagi Otomatis**: Jika terjadi kesalahan jaringan, manajer akan mencoba memulihkan unduhan secara otomatis.

## Lokasi Penyimpanan
Secara bawaan (*default*), semua chapter disimpan di folder `Yomic/Downloads` di dalam direktori AppData sistem Anda. Anda dapat mengelola atau menghapus file-file ini dari menu Pengaturan Penyimpanan.
