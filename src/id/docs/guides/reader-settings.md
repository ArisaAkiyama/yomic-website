# Pengaturan Pembaca

Yomic menyediakan pengalaman membaca yang sangat fleksibel dan dapat disesuaikan. Anda dapat mengakses pengaturan pembaca dengan membuka bab manga apa saja lalu mengeklik ikon pengaturan.

## Mode Membaca

Anda dapat memilih beberapa mode membaca berikut sesuai dengan keinginan Anda:

- **Webtoon (Gulir Vertikal):** Gambar ditampilkan terus-menerus dari atas ke bawah. Sangat ideal untuk membaca komik jenis webtoon.
- **Halaman Tunggal:** Menampilkan satu halaman dalam satu waktu. Klik sisi layar atau gunakan tombol panah pada keyboard untuk berpindah halaman.
- **Halaman Ganda:** Menampilkan dua halaman berdampingan secara horizontal, mirip seperti membaca buku komik cetak fisik.

## Tampilan

- **Warna Latar Belakang:** Mengubah warna latar belakang di sela-sela halaman (misalnya Hitam, Putih, atau warna Kustom).
- **Tipe Skala:** Mengatur bagaimana gambar menyesuaikan diri dengan ukuran layar (Pas Lebar, Pas Tinggi, atau Ukuran Asli).

## Filter

Anda dapat menerapkan filter warna khusus pada layar pembaca, yang sangat berguna untuk membaca di malam hari atau untuk mengurangi kelelahan mata:
- Mengatur tingkat kecerahan (*brightness*) dan kontras (*contrast*).
- Menerapkan efek filter warna (misalnya filter warna hangat/amber untuk kenyamanan membaca di malam hari).
