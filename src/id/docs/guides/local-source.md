# Sumber Lokal (Local Source)

**Status:** 🚧 Saat ini sedang dalam pengembangan

Saat ini, Yomic bertindak sebagai *browser* terpusat untuk katalog daring (online) dan memungkinkan Anda untuk mengunduh chapter-chapter tersebut agar dapat dibaca secara luring (*offline*). Namun, dukungan *native* untuk mengimpor file `.cbz`, `.cbr`, `.zip`, atau folder gambar mentah milik Anda sendiri sebagai ekstensi "Sumber Lokal" khusus sedang direncanakan untuk pembaruan mendatang.

### Bagaimana cara membaca file luring (offline) saat ini?
Jika Anda telah mengunduh chapter menggunakan [Manajer Unduhan](../browse/downloads.md) milik Yomic, Anda dapat membacanya sepenuhnya secara luring! Yomic secara otomatis mendeteksi jika sebuah chapter ada di dalam folder `Yomic/Downloads` lokal Anda dan akan memuat gambar-gambar tersebut langsung dari perangkat keras Anda tanpa membuat permintaan jaringan apa pun.
