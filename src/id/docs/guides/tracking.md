# Pelacakan (Tracking)

**Status:** 🚧 Saat ini sedang dalam pengembangan

Mengintegrasikan Yomic dengan layanan pelacakan pihak ketiga (seperti MyAnimeList, AniList, Kitsu, dan MangaUpdates) adalah fitur yang sangat banyak diminta dan saat ini berada di peta jalan (*roadmap*) kami untuk pembaruan mendatang.

Setelah diimplementasikan, fitur ini akan memungkinkan Anda untuk:
- Menyinkronkan kemajuan bacaan Anda secara otomatis dengan daftar daring (online) Anda.
- Melihat skor komunitas dan metadata terperinci.
- Mengimpor daftar daring Anda yang sudah ada secara langsung ke dalam perpustakaan Yomic.

Untuk saat ini, Yomic bergantung sepenuhnya pada basis data internalnya untuk melacak riwayat bacaan dan kemajuan perpustakaan Anda. Kami merekomendasikan untuk secara rutin menghasilkan [Pencadangan (Backup)](./backups.md) untuk memastikan progres Anda tidak pernah hilang!
