# FAQ Perpustakaan

Pelajari cara mengelola koleksi manga lokal Anda dengan efektif.

### Bagaimana cara menambahkan manga ke perpustakaan saya?
Saat menelusuri manga di suatu sumber, cukup klik tombol **"Tambahkan ke Perpustakaan"** di halaman Detail Manga (biasanya dilambangkan dengan ikon hati atau markah buku). Manga tersebut akan langsung muncul di tab Perpustakaan Anda.

### Mengapa beberapa manga menampilkan lencana "BARU" ("NEW")?
Lencana **BARU** akan muncul secara otomatis ketika Yomic mendeteksi bahwa sebuah manga di perpustakaan Anda telah merilis chapter baru yang belum Anda baca. Fitur ini menggunakan deteksi pintar untuk membandingkan progres bacaan Anda dengan chapter terbaru yang tersedia.
Lencana ini akan hilang dengan sendirinya setelah Anda membuka chapter tersebut atau menandainya sebagai sudah dibaca.

### Mengapa perpustakaan saya tidak melakukan pembaruan (update) secara otomatis?
Untuk menghemat kuota internet dan daya pemrosesan, pembaruan otomatis global ditangani oleh algoritma **Pembaruan Pintar (Smart Updates)** kami. 
Jika sebuah manga sedang hiatus atau sudah lama sekali tidak mendapatkan pembaruan, Yomic mungkin akan melewatinya selama siklus pembaruan global. Anda selalu bisa memaksa pembaruan dengan mengusap layar ke bawah (pull down) atau mengeklik tombol segarkan di halaman Detail Manga.

### Bisakah saya mengatur perpustakaan ke dalam folder?
Tentu saja! Mulai dari v1.5.0, Anda bisa menggunakan **Kategori** untuk mengatur perpustakaan Anda.
- Klik tombol **Kelola Kategori** di Perpustakaan Anda untuk membuat folder khusus.
- Anda dapat menetapkan sebuah manga ke dalam satu atau beberapa kategori sekaligus.
- Anda bahkan dapat mengecualikan kategori tertentu (seperti "Tamat" atau "Hiatus") dari siklus pembaruan otomatis global untuk menghemat waktu!
