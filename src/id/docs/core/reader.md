# Pembaca (Reader)

Yomic menyediakan pembaca native yang tangguh, dirancang khusus untuk platform desktop guna memberikan pengalaman membaca yang paling nyaman.

## Mode Membaca

Anda dapat menyesuaikan bagaimana chapter ditampilkan bergantung pada jenis komik yang sedang Anda baca:

- **Mode Webtoon**: Dirancang khusus untuk komik memanjang (Manhwa/Webtoon). Gambar dimuat dengan mulus dalam gulungan vertikal (scroll) tanpa jeda antar halaman.
- **Mode Manga (Dual-Page Spread)**: Mulai dari v1.5.0, Yomic mendukung penayangan dua halaman secara berdampingan! Mode ini memanfaatkan navigasi **Kanan-ke-Kiri** (Right-to-Left) untuk meniru pengalaman autentik membaca manga fisik Jepang.

## Penyesuaian dan Penskalaan (FitScale)
Anda dapat menyesuaikan ukuran skala gambar dengan menggunakan opsi **FitScale**.
- **Fit Width**: Menyesuaikan skala gambar agar pas secara horizontal (bawaan untuk Webtoon).
- **Fit Height (90%)**: Menyesuaikan skala gambar sehingga seluruh halaman terlihat secara vertikal dengan sedikit margin, mencegah gambar terpotong.

## Navigasi
Pembaca kami dioptimalkan sepenuhnya untuk penggunaan desktop:
- **Tombol Panah (Arrow Keys)**: Gunakan panah Kiri/Kanan atau Atas/Bawah untuk menggeser gambar dan berpindah antar halaman.
- **Transisi Otomatis**: Saat mencapai akhir chapter dan menekan tombol "Selanjutnya", Yomic akan otomatis memuat dan mengindeks chapter berikutnya berdasarkan pengaturan Urutan (SortOrder) Perpustakaan Anda.
